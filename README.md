## 🎯 Aperçu

Cette application permet aux passionnés de football de suivre en temps réel :
- **13+ compétitions internationales** (Premier League, La Liga, Champions League, etc.)
- **Matchs en direct et à venir** avec scores et statistiques
- **Classements détaillés** des équipes par compétition
- **Informations complètes** sur les équipes et joueurs

L'application utilise l'API gratuite de [Football-Data.org](https://www.football-data.org/) pour récupérer les données.
